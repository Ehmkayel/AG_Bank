### Introduction

A modern, responsive mortgage calculator web application that provides users with an intuitive digital credit experience.

## Features

- Interactive Mortgage Calculator: Real-time payment calculations with sliders
- Responsive Design: Works seamlessly on desktop and mobile devices
- User-Friendly Interface: Clean, modern design with intuitive controls
- Multiple Loan Options: 5/1 ARM, 15-year fixed, and 30-year fixed mortgage options
- Direct input fields with instant feedback
- Synchronized sliders and input fields
- Input validation with helpful prompts

## Getting Started

To get started, simply click on the link [AG Bank](https://ag-bank.vercel.app/)

## User Guide

- Set Purchase Price
- Use the slider 
- Range: $50,000 to $2,500,000
- set Down Payment
- Minimum 4% of purchase price
- Select Loan Period
